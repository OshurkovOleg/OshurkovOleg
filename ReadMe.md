# 💫 About Me:
Специализируюсь на backend-разработке корпоративных веб-приложений на Java. Основной стек: Java 11, Spring Boot, Spring MVC, Spring Data, PostgreSQL, Kafka, Liquibase, GitLab CI/CD. Участвовал в развитии бизнес-логики, проектировании API и интеграции с внешними системами, занимался оптимизацией производительности и поддержкой production-среды. Есть опыт работы с modular monolith и взаимодействия с микросервисами.
